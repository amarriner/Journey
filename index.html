Flora and Fauna

This page hosts my entry for NaNoGenMo 2014. My novel is a first-person narrative that details a botanist/zooligist's journey through a series of mazes. They talk about the directions they take as well as the flowers and animals they find.

Each "floor" represents a maze, and is written as a discrete chapter. Every floor is a randomly generated maze that the narrator must traverse and find the exit. There are three versions of output: plain text, HTML, and PDF. The PDF file isn't generated via the script, but the text and HTML versions are. In the HTML nad PDF versions, the script generates an image of each floor so you can see the maze and the path the narrator took through that floor.
